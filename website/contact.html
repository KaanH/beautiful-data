Contact Information

We welcome participation in the beautiful-data project on all fronts. We're always looking for data science fanatics regardless of their background. Alternatively, if your organization is in need of a tool for doing big data work, please also let us know how we can help.

For more information on BD and how you can help us grow or how we can help your organization, please send email to johnjtran@acm.org. The project's source code repository, wiki, and issue tracking can be found on the github page: https://github.com/lazy-bones/beautiful-data.
