About

Beautiful Data (BD) is an Open Source project that focuses on a large-scale data processing and visualization. Its ultimate goal is to provide researchers, business analysts and students a framework for doing "big data" in relatively short order.

Contact

For more information on BD and how you can help, please send email to johnjtran@acm.org. The project's source code repository, wiki, and issue tracking can be found on the github page: https://github.com/lazy-bones/beautiful-data.
